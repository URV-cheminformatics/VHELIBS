VHELIBS

Validation HElper for LIgands and Binding Sites

Download .jar View on GitHub

VHELIBS aims to ease the validation of binding site and ligand structure models using their electron density maps, in a semi-automatic way, classifying them based on their RSR values as 'good' (with an RSR below a certain value), 'bad' (with an RSR over a certain value) or 'dubious' (with an RSR values between the aforementioned values). After performing the automatic analysis and classification of a target's binding site and ligand, it then enables the user to graphically review and compare them with their EDM in order to make it easier to properly classify any 'dubious'-labelled structure or re-classify any other structure based on actual graphical fitting of the EDM with the model.

VHELIBS aims to enable non-crystallographers and users with little or no crystallography knowledge to easily validate and use reliable structures in drug discovery and development. To that end, it features a Graphical User Interface (GUI) which makes it easy to tune the parameters for more advanced users, but also comes with carefully chosen default values that are valid for most situations.
